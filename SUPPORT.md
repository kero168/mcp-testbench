# Support

Thanks for using `mcp-testbench`! Here is how to get help.

## Questions & usage help

- Check the [README](./README.md) and [docs/](./docs/) first.
- Open a [GitHub Discussion](https://github.com/kero168/mcp-testbench/discussions)
  for questions, ideas, and "how do I…" topics.

## Bugs

- Open a [bug report](https://github.com/kero168/mcp-testbench/issues/new/choose)
  with the reproduction template. Please include your OS, Node version, the
  exact command you ran, and the server command/URL you were testing.

## Security issues

- **Do not open a public issue.** Follow [SECURITY.md](./SECURITY.md).

## Response times

Target: first response within 48 hours for issues, 72 hours for PR review
(median). This is a volunteer-maintained project; thanks for your patience.
